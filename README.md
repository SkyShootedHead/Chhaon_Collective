# Chhaon Collective

A modern, collaborative web application built with React, TypeScript, and Vite. Chhaon Collective provides a beautiful, responsive interface using shadcn/ui components and Tailwind CSS.

## Features

- ⚡ **Lightning-Fast Development** - Powered by Vite for instant HMR and optimized builds
- 🎨 **Beautiful UI Components** - Comprehensive shadcn/ui component library with Radix UI primitives
- 📱 **Fully Responsive** - Mobile-first design with Tailwind CSS
- 🎭 **Theme Support** - Dark and light mode support with next-themes
- 📊 **Rich Data Visualization** - Charts and graphs with Recharts
- 🔄 **State Management** - TanStack React Query for efficient data fetching
- ✅ **Form Handling** - React Hook Form with Zod validation
- 🧪 **Testing Ready** - Vitest and Playwright for comprehensive testing
- 📦 **Type-Safe** - Full TypeScript support throughout the codebase

## Tech Stack

### Frontend
- **Framework:** React 18.3
- **Language:** TypeScript 5.8
- **Build Tool:** Vite 5.4
- **Styling:** Tailwind CSS 3.4 + PostCSS
- **UI Components:** shadcn/ui with Radix UI
- **Routing:** React Router DOM 6.30
- **State Management:** TanStack React Query 5.83

### Form & Validation
- **Form Management:** React Hook Form 7.61
- **Form Resolvers:** @hookform/resolvers 3.10
- **Validation:** Zod 3.25

### UI Enhancements
- **Animations:** Framer Motion 12.38
- **Icons:** Lucide React 0.462
- **Notifications:** Sonner 1.7
- **Date Picking:** React Day Picker 8.10
- **Carousel:** Embla Carousel React 8.6
- **Charts:** Recharts 2.15

### Development Tools
- **Testing:** Vitest 3.2, Playwright 1.57
- **Linting:** ESLint 9.32 with TypeScript support
- **Type Checking:** TypeScript 5.8
